Enumerable#cycle

April 4, 2014

cycle(n=nil) {|obj| block } -> nil
cycle(n=nil) -> an_enumerator
Enumerable#cycle calls block for each element repeadedly.

Arguments

If no argement is given, it calls the block on each element forever. Otherwise, the block is called n times.

Returned Object

If block is givien it returns the block defines. Otherwise, it returns an enumerator resulted from the method.

Example

Let's say that we want to find out the day of a certain date of the month and we want to create a method that does that for us. This is easily done using .cycle.

# Two parameters. The day of the first date of the month and the date of the day we're looking for.
def day_of_month (first_of_month, date)
days = %w(Monday Tuesday Wednesday Thursday Friday Saturday Sunday)
# Rotate the days array so the day of the first date of the month becomes the first element of the array
days.rotate!(days.index(first_of_month))
# Repeat the array for five times and return the date-th element of it.
days.cycle(5).to_a.fetch(date-1)
end

day_of_month('Tuesday', 28) => 'Monday'
