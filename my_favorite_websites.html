Amazon.com

I think amazon.com is my absolute favorite. Their home page is a little bit busy since it's full of advertizements and product pictures but it is still very easy and intuitive to get to where I want to go. Obviously, I use the website to shop, so when I'm on amazon.com, I would be looking for a paticular product or checking out my wish list to see if I'd like to purchase something from that list.
By using the search bar or the department list, I am always able to find what I'm looking for. If I'm looking for a sepecific product, I use the search bar. It has an autocomplete function so i don't have to type the whole name, and it also gives me the option to search in a paticulat depatment if I want to narrow down the list.
The Department list is awesome too. It is full of ( not too many ) sublists so it is usually easy for me to pinpoint the things I'm looking for from the list.
Thier wish list also one of my favorite features. I get to make multiple wish lists and it is so easy to edit them.
