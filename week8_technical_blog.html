How PCs Work

May 4, 2014

We all use PCs every day, but do we know how they work? I sure don’t! So let's find out…

What’s inside of a PC? Let’s learn about its parts.

  • Case

    PC needs something to cover the rest of the components.

  • Motherboard

    All the other components go through here. Some important components (including CMOS) are directly connected to the motherboard.

  • Power supply

    PC needs this to work!

  • CPU (Central Processing Unit)

    Also called professor which makes a computer a computer. This is the heart of a PC and determines how the hardware and software perform.

  • RAM (Random Access Memory)

    Helps the CPU’s performance by temporality holding information while the CPU is being processed.

  • Cooling Devises

    Cools off the PC.

  • Cables

    Connects the components together.

  • Graphic Components

    Helps CPU process complex graphics data.

  • Ports

    Places where you can plug in cables includeing , USB and Ethernet.

  • Peripherals

    Monitor, keyboard, mouse, speaker, printer, etc…

  • Expansion slots

    A place extra components (called cards) can go.

Starting a PC

  1. Push the button!
  2. Start button activates the power supply in the PC which sends the power to the motherboard and other components.
  3. The PC performs the POST(Power-On Self-Test). The POST is a small computer program within the BIOS that checks for hardware failures. A beep that you hear when you start a PC means that everything went ok, if you hear multiple beeps, something went wrong
  4. PC displays the status of the booting process including BIOS manufacturer and revision, professor specs, the amount of RAM installed, and the drives detected.
  5. The BIOS attempts to access the first sector of the drive which contains the operating system.
  6. The BIOS confirms there’s a bootstrap loafer, and loads that boot loader into RAM. The boot loader is a small program designed to find and launch the OS.
  7. Once the boot loader is in memory, the BIOS hands over its work to the boot loader, which starts loading the OS into memory.
  8. Once the boot loader is done with its work, the OS takes over the control of the PC.

Now That the PC is started properly, the OS can do its jobs which include…

  • Processor management
  • Memory management
  • Device management
  • Storage management
  • Application interface
